Native WooCommerce Analytics reports backed by subscription analytics lookup tables.

== Description ==

Additional Subscriptions Analytics is a private prototype for modern WooCommerce
Subscriptions analytics. It will maintain subscription analytics lookup tables
and expose native WooCommerce Analytics reports, beginning with Upcoming
Renewals.

== Installation ==

1. Install WooCommerce and WooCommerce Subscriptions.
2. Upload or clone this plugin.
3. Run Composer and npm install steps for development builds.
4. Activate the plugin from WordPress admin.
